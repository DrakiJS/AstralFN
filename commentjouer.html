Tutoriel d'Installation du Launcher

Étape 1 : Téléchargement du Launcher

Commencez par télécharger le launcher à partir du site officiel. Vous pouvez le trouver sur la page de téléchargement de notre site web.

Étape 2 : Installation du Launcher

prochainement...

Étape 3 : Configuration du Launcher

Rends toi sur le discord et effectue /create avec ton pseudo , mail ET mot de passe.. ce sont tes identifiants ne les perds pas ! (nous avons accès a ton email et pseudo mais pas ton mot de passe)

Étape 4 : Téléchargement de la saison (Fn build)

Après avoir configuré votre compte, vous pouvez commencer à télécharger les saisons disponible dans le launcher. Parcourez la bibliothèque de jeux et cliquez sur le bouton de téléchargement pour les installer sur votre système. ou via ce lien https://cdn.fnbuilds.services/12.50.zip

Étape 5 : Lancement du jeu

Une fois le téléchargement terminé, vous pouvez lancer vos saisons directement à partir du launcher. Recherchez le jeu que vous souhaitez jouer dans votre bibliothèque et cliquez sur le bouton de launch pour lancer le jeu.
